Resistograph

Resistograph data processing module.

This module provides a class to handle resistograph data processing, including reading, preprocessing, and removing global linear trends from the data. It also includes a function to visualize the resistograph data as colored bars on a tomogram image.

This code is part of the Tomotree project.

preprocess_data()

Preprocess the resistograph data by applying various transformations.

This method reads the data, sets initial and final rows as NaN, applies smoothing if enabled, and removes values below a minimum threshold.

read_resistograph_data()

Read resistograph data from files in the specified directory and process it.

The method reads all files with the extension .dpa, extracts the data section, and calculates the square root of the values. The processed data is stored in a pandas DataFrame.

remove_global_linear_trend()

Remove the global linear trend from the resistograph data.

This method calculates the linear trend across all columns and subtracts it from the data to normalize the values.
