tomotree.resistograph.add_resistograph_to_image

tomotree.resistograph.add_resistograph_to_image(data, tomogram, ax=None, cax=None, min=100, max=200, zorder=100, columns=None, linewidth=16, meter_length=20, step=300, **kwargs)

Plot resistograph data as series of colored bars on the picture with tomogram.

This function visualizes resistograph data by overlaying it as colored bars on a tomogram image. The bars are color-coded based on the resistograph values, and a color bar is added to indicate the value range. Additionally, a scale indicator is drawn on the image.

Parameters:

  • data (pandas.DataFrame) – A DataFrame containing resistograph data. Each column represents a resistograph measurement for a specific node.

  • tomogram (object) – An object representing the tomogram. It must have a get_node_by_number method that returns nodes with coordinates attributes.

  • ax (matplotlib.axes.Axes, optional) – The axes on which to plot the resistograph data. If None, a new figure and axes are created.

  • cax (matplotlib.axes.Axes, optional) – The axes on which to plot the color bar. If None, the color bar is added to the main axes.

  • min (float, optional) – The minimum value for the color normalization. Default is 100.

  • max (float, optional) – The maximum value for the color normalization. Default is 200.

  • zorder (int, optional) – The z-order for the plotted elements. Default is 100.

  • columns (list, optional) – A list of column indices from the DataFrame to plot. If None, all columns are used.

  • linewidth (float, optional) – The width of the colored bars. Default is 16.

  • meter_length (float, optional) – The length of the scale indicator in meters. Default is 20.

  • step (int, optional) – The step size for down-sampling the resistograph data. Default is 300.

  • **kwargs (dict) – Additional keyword arguments for customization.

Returns:

ax – The axes with the plotted resistograph data.

Return type:

matplotlib.axes.Axes

Notes

  • The function assumes that the tomogram nodes are numbered sequentially and that their coordinates are accessible via the get_node_by_number method.

  • The scale indicator is drawn at the bottom-left corner of the plot, with labels “5cm” and “10cm”.

  • Missing or NaN values in the resistograph data are ignored during plotting.
